// Learning Roadmap

Python学習ロードマップ

銀行のデータ分析・リスク管理職を目指すための学習プラン。
SQLと並行しながら、約3ヶ月で実務レベルの基礎を身につけます。

🎯 目標: 銀行データ分析・リスク管理職
Week 1-2
Python基礎文法
まずはPythonの書き方に慣れる。完璧に覚えようとせず、「こういうことができる」を知るだけでOK。
変数・型 if / for / while 関数 def リスト・辞書 ファイル読み書き
// ゴール
簡単な計算やデータ処理のスクリプトが自力で書ける
Week 3-5
pandas でデータ加工
ここが一番大事なフェーズ。CSVやExcelを読み込んで、フィルタ・集計・結合ができるようになる。SQLでやったことがPythonでもできる感覚がつかめます。
pd.read_csv フィルタリング groupby / agg merge（≒ JOIN） 欠損値処理 pivot_table
// ゴール
生データのCSVを渡されて「月別売上を集計して」と言われたら対応できる
Week 6-7
データ可視化
数字をグラフにして「見せる力」を身につける。銀行のレポートやプレゼンで必ず求められるスキル。
matplotlib seaborn 棒グラフ・折れ線 ヒートマップ 散布図
// ゴール
データから意味のあるグラフを選んで作成し、傾向を説明できる
Week 8-10
統計基礎 + 実践プロジェクト
リスク管理に直結する統計の基礎と、実際のデータセットを使ったミニプロジェクトに挑戦。ポートフォリオにもなります。
平均・中央値・分散 相関分析 正規分布 numpy ミニプロジェクト
// ゴール
「顧客データを分析してリスク傾向をレポートして」に対応できる
Week 11-12
就活準備 + 応用
学んだことを整理して、ポートフォリオや面接の準備。余裕があればPower BIやJupyter Notebookでの報告書作成も。
Jupyter Notebook ポートフォリオ整理 Power BI 入門 面接対策
// ゴール
「Pythonでこういう分析ができます」と具体例つきで説明できる
📌 SQLは並行して進める：今の課題集を1日1問ペースで消化 → 終わったらGROUP BY・COUNT・SUMの練習を少しだけ追加。合計2〜3週間で完了でOK。
